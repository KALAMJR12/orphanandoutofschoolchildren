# Sheik Ahmad Tijjani Umar Foundation - 80th Birthday Initiative Website

## Overview
Professional website celebrating Alhaji Atiku Abubakar's (GCON) 80th birthday and showcasing the Sheik Ahmad Tijjani Umar Foundation's education initiative for orphans and out-of-school children from primary to secondary level.

## Project Architecture
- **Technology Stack**: HTML5, CSS3, Vanilla JavaScript
- **Design**: Sharp, modern design using foundation's official brand colors (green and orange)
- **Server**: Python HTTP server on port 5000
- **Responsive**: Fully responsive design for all devices

## Recent Changes
- November 08, 2024: Initial website creation
  - Created responsive HTML structure with hero, celebration, initiative, impact, foundation, CTA, and contact sections
  - Implemented CSS with foundation's official brand colors (green and orange)
  - Added JavaScript for smooth scrolling, animations, and interactive elements
  - Set up Python HTTP server workflow
  - Removed all donation-related content per foundation requirements

## Key Features
- Hero section with 80th birthday celebration message
- About the initiative section highlighting orphan and out-of-school children education program
- Foundation information with mission and values
- Impact statistics with animated counters
- Call-to-action section for engagement and volunteering
- Contact form and information
- Smooth scroll animations and interactive elements
